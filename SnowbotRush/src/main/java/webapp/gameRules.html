HOW TO PLAY SNOWBOT RUSH

Instructions:

Step 1: Enter name to start game and select level of difficulty. Then select "Start Game".

Step 2: Wait for second player to join. (Second player should follow step 1 as well, on another window/tab to join the same game).

Step 3: When second player has joined, the game board would display to players.

Step 4: Players can click any box they want in the game board.

Step 5: If the box the player clicked is a snowflake, they gain a life.

If the box the player clicked is a snow enemy, the player would lose a life.

If the box the player clicked is a coin, jewel, or a diamond, the player would gain 10, 25, or 50 points respectively.

If the box the player clicked is an icicle, the player would be frozen for 3 seconds, which means that the player can not play for the next 3 seconds of the game.

If the box the player clicked is a X, the player gets notified that the other player has claimed the object in the box.

Step 6: The players start with 7 lives, and when they lose them all, the player with the highest score wins

Features of the Game:

1. A multiplayer game where only max two players can play a game, but multiple games can be played at a time.

2. The game messages displayed at the top of the board shows the actions of the players.

3. The chat box at the side of the game allows the players to interact with each other through the server.

4. The high score table displays the top 5 player high scores from highest to lowest that gets refreshed every minute.

5. The game screen displays the game board once both players join the game.

Enjoy!
